Simple Python Calculator App

This is a basic terminal calculator with add, subtract, multiply, and divide functions.

How to run locally:
py calculator.py

How to build Docker image:
docker build -t ahmedalsheikh23/calculator-app .

How to run Docker container:
docker run -it ahmedalsheikh23/calculator-app
